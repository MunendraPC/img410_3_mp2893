# Project 3 - Raycasting

This program implements a simple non-recursive ray casting renderer in C++. 
It reads a scene file, computes ray intersections with spheres and planes, 
and writes the result to a PPM (P3) image file.

# Authors

Liam O'Donnell - lko27@nau.edu, Munendra Pratap Choudhary - mp2893@nau.edu

# Usage

Build the program:

make

Run the program:

./raycast width height input.scene output.ppm
